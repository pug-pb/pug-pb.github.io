Parceria com a Jetbrains

A partir do próximo meetup do PUG-PB (Eventos), passaremos a sortear licenças para utilização em qualquer produto da Jetbrains.

A empresa conta em seu porfólio com excelentes ferramentas de desenvolvimento, a exemplo do ótimo PyCharm, IDE para desenvolvimento na linguagem de programação Python.

O PyCharm, que funciona nas plataformas Windows, Linux e Mac, tem uma versão free e open source chamada Community, que pode ser baixada aqui.

As licenças que serão sorteadas poderão ser utilizadas para a versão PyCharm Professional, que tem alguns diferencias da versão gratuita:

  • Web development
  • Python web frameworks
  • Python Profiler
  • Remote development capabilities
  • Database & SQL support
