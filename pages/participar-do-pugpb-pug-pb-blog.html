Artigos para o blog

O blog terá atualização, no mínimo, semanal. Mande suas colaborações de acordo com as instruções a seguir.

Pelo Github

  1. Clone (ou fork) o repositório

  2. No diretório "blog", crie um novo arquivo com extensão ".md". Copie o conteúdo do arquivo "padrao-artigo.md" no arquivo criado, faça suas modificações e salve.

  3. Solicite "pull request" com a colaboração no repositório principal.

  4. Aguarde a revisão dos moderadores. Se estiver tudo OK seu artigo será publicado.

Pelo Fórum

  1. Poste o conteúdo do seu artigo no fórum, incluindo no título da postagem a marca [BLOG].

  2. Aguarde a revisão dos moderadores. Se estiver tudo OK seu artigo será publicado.
